Cahier des charges — ENTIDR
1. Présentation du projet
ENTIDR est une plateforme modulaire d'entreprise basée sur Odoo pour le backend et sa base de données, avec un frontend en React + Tailwind CSS.
Le projet doit permettre de gérer des addons (ex. : RH, Finance, etc.), qui contiendront leurs propres vues, modèles, contrôleurs, etc.

Le but est d'avoir une plateforme évolutive avec :

Installation / désinstallation d'addons via l'interface.

Organisation claire dans un seul dossier src/.

Utilisation de Tailwind CSS pour tout le design.

Une structure frontend robuste, avec une séparation nette des vues de base et des addons.

2. Stack Technique
Frontend : React + TypeScript + Tailwind CSS

Backend : Odoo

Base de données : PostgreSQL (via Odoo)

Structure : Monorepo avec un seul dossier src/

Gestion d’addons : système inspiré d’Odoo

4. Fonctionnalités principales
🔐 Authentification : Login, inscription, récupération de mot de passe.

🧩 Gestion des Addons : Installation, activation/désactivation, suppression depuis l’interface.

🗂️ Gestion des modules HR et Finance :

HR : gestion des congés, employés, feuilles de temps.

Finance : facturation, dépenses, rapports.

⚙️ Paramétrage global : Configuration des préférences systèmes via l’interface admin.

🎨 Design : UI moderne avec Tailwind CSS, responsive mobile et desktop.
